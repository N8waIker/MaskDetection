# MaskDetection

In this kernel I was able to designt a real time tracking mask detector. It distinguishes between not wearing a mask, wearing a mask properly and wearing a mask wrong. As always shout out to https://www.youtube.com/channel/UCHXa4OpASJEwrHrLeIzw7Yg for making this possible!
